## 代码结构
```cpp
printf("server says: [%u] %.*s\n", rescode, len - 4, &rbuf[8]);
```

## 各部分解析

### 1. 固定字符串部分
```cpp
"server says: [%u] %.*s\n"
```
- `"server says: "` - 固定文本
- `[%u]` - 格式化输出一个无符号整数
- `" "` - 空格
- `%.*s` - 特殊格式的字符串输出
- `\n` - 换行符

### 2. 参数解析

```cpp
rescode,          // 对应 %u - 无符号整数
len - 4,          // 对应 *  - 字符串长度
&rbuf[8]          // 对应 s  - 字符串起始地址
```

### 3. `%.*s` 的特殊含义

`%.*s` 是printf的一个特殊格式：
- 第一个 `*` 表示字符串长度从参数中获取
- `s` 表示输出字符串
- 格式：`%.*s` = 输出指定长度的字符串

等价于：
```cpp
// 假设长度为 5，字符串指针为 str
printf("%.*s", 5, str);  // 只输出str的前5个字符
//
